How to use

Chat Command

Configure a command (e.g. !IQ) in the General settings. The widget will generate a random IQ score for the user who typed it. Use !IQ @username to target a specific user.

or
Channel Point

Set the exact name of your Channel Points redemption in General settings. If the redemption message contains a username, the result will be generated for that user instead.

How to choose your layout

In the overlay editor, go to the left panel under the widget name and open the Layers tab. There, you can select the layout you want to use. Make the other layout invisible by clicking the eye icon next to it.

SE BOT Integration

These are optional settings. To use them, you need to connect the StreamElements bot to your Twitch channel and assign it the moderator role. This can be done in the StreamElements dashboard on the right side.

In this tab, you will find three fields:

JWT Token
Channel ID
Bot Message

All of these fields are required for the feature to work correctly.

You can find the first two values by clicking your avatar icon in StreamElements and selecting Channel Settings. From there, copy your Account ID and JWT Token into the appropriate fields.

Important: Do not share these credentials with anyone.

The last field, Bot Message, defines the message that the bot will send in chat. It supports two variables:

{{username}} which is replaced with the user's name
{{value}} which is replaced with the generated IQ value

General Settings

Hide in idle (checkbox)
When enabled, the widget remains invisible while no event is active.
Trigger method (selection field)
Choose how the widget is activated.
Channel Point Name
Enter the exact name of the Channel Points redemption.
Chat Command
Enter the command that users need to type in chat.
Min IQ and Max IQ
Define the minimum and maximum IQ values that can be randomly generated.
Display Results
Set how long, in seconds, the result will be visible.
Placeholder
The text displayed when the widget is visible but no event is currently running.
Sound (optional)
An optional field where you can add a sound effect that will play during the randomization.

Text Settings

Username Font Size
Sets the font size for the username.
Username Font Weight
Sets the font thickness for the username.
Value Font Size
Sets the font size for the IQ value.
Value Font Weight
Sets the font thickness for the IQ value.

Color Settings

In the Color tab, you can set the colors for individual elements of the widget.
